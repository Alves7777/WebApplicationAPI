
---

### **Opçăo 2: Configurar para aceitar conexőes externas**

Edite: `Properties/launchSettings.json`

```json
{
  "profiles": {
    "http": {
      "commandName": "Project",
      "dotnetRunMessages": true,
      "launchBrowser": true,
      "applicationUrl": "http://0.0.0.0:5000",
      "environmentVariables": {
        "ASPNETCORE_ENVIRONMENT": "Development"
      }
    }
  }
}
```

Depois rode:
```powershell
dotnet run --launch-profile http
```

## ?? **IMPORTANTE - CORS:**

O backend **JÁ ESTÁ CONFIGURADO** para aceitar requisiçőes do frontend!

`Program.cs` (Backend):
```csharp
builder.Services.AddCors(options =>
{
    options.AddPolicy("AllowFrontend", policy =>
    {
        policy.WithOrigins("http://localhost:5173", "http://localhost:3000", "http://localhost:5000", "https://localhost:5001")
              .AllowAnyMethod()
              .AllowAnyHeader();
    });
});

app.UseCors("AllowFrontend");
```
